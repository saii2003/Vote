# Vote目錄檔案說明

![index text](https://saii2003.github.io/Introduction/intro_vote/image/1.jpg)

<pre><code>### 使用技術</code></pre>
- 後端技術：ASP.NET C#
- 前端技術：HTML、CSS
- 資料庫：MSSQL 2008
- 架構：Web Form

<pre><code>### App_Code</code></pre>
- DBbase.cs ：資料庫連線基本設定
- Vote.ce ：投票頁面相關類別
- Member.cs ：會員頁面相關類別

<pre><code>### root</code></pre>
- index.aspx ：首頁
- vote.aspx ：投票頁面
- vote_add.aspx ：新增投票頁面
- vote_end.aspx ：投票結果頁面

<pre><code>### Member</code></pre>
- forget_of.aspx ：忘記密碼訊息
- forget_pw.aspx ：忘記密碼
- member_add.aspx ：加入會員
- new_pw.aspx ：變更密碼
